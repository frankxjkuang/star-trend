# star-trend
Rendering a trend chart for a repo.

# Usage

```
go run main.go
```

Browser access to `127.0.0.1:9000`

Example like:

[![Stargazers over time](http://127.0.0.1:9000/ frankxjkuang/star-trend)](http://127.0.0.1:9000/ frankxjkuang/star-trend)
